# Contributing

Thank you for improving the Exhibitor Intelligence Playbook.

## How to contribute

- Open an issue to suggest improvements or new sections.
- Submit a pull request with edits to the playbook chapters.

## Style guidelines

- Keep the content actionable and B2B-focused.
- Use clear headings and bullet points for quick scanning.
- Avoid vendor bias except in the tools section.
- Include sources or rationale when adding benchmarks or metrics.

## Review process

All contributions are reviewed for accuracy, clarity, and relevance to trade show ROI outcomes.
